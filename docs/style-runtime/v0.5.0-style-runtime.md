# v0.5.0 Style Runtime

Adds:
- style section runtime (INTRO1, VAR1, VAR2, FILL1, ENDING1)
- phrase event playback with quantized timing
- chord transposition (DRUMS unaffected, bass/accompaniment transposed)
- bar-quantized section switching
- live style runtime panel with test controls

This milestone starts real arranger-style behavior.

## Features

### Style Sections
- **INTRO1** — 4-step introduction
- **VAR1** — Basic variation
- **VAR2** — Busier variation with more drums
- **FILL1** — Drum fill
- **ENDING1** — Ending section

### Phrase Playback
- 16 steps per bar
- Multi-track events (DRUMS, BASS, ACC1)
- Velocity control per note
- Step-based triggering

### Chord Transposition
- Detects chord root (C, D, A, etc.)
- Transposes BASS and ACC1 notes
- DRUMS unaffected
- Real-time chord changes

### Section Switching
- Queue sections for smooth bar-aligned transitions
- Apply on bar boundary (not mid-bar)
- No audio artifacts

## Usage

```javascript
import { createArrangerStyleRuntime } from "./arrangerStyleRuntime.js";

const runtime = createArrangerStyleRuntime();

runtime.start();
runtime.queueSection("VAR2");
runtime.pulse();  // Advance one step
runtime.pulse();  // Advance one step
// ... 16 pulses = 1 bar
runtime.setChord("D major");

const state = runtime.snapshot();
// {
//   running: true,
//   bar: 1,
//   beat: 1,
//   chord: "D major",
//   renderedEvents: [ { events: [...], renderedAt: ... } ]
// }
```

## Test Component

StyleRuntimePanel provides:
- Start/Stop buttons
- Pulse (single step) and 1 Bar buttons
- Section buttons (INTRO1, VAR1, VAR2, FILL1, ENDING1)
- Chord buttons (C, D, Am)
- Live JSON state output

## Architecture

```
StyleRuntimePanel (React)
  └─ createArrangerStyleRuntime()
     ├─ createStyleSectionRuntime()
     │  └─ 5 sections with phrase events
     └─ transposeStyleEvents()
        └─ Chord-based transposition
```

State includes:
- Running status
- Current bar, beat, step
- Queued section (for next bar)
- Rendered events history (64 events)
- Current chord
- Section library
