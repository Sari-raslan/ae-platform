# Unified KA Mission

This batch adds one orchestrated mission layer that uses all KA agents:

- KA Parser Assistant
- KA MIDI Assistant
- KA UI Assistant
- KA Cloud Assistant
- KA Mobile Assistant
- KA AI Assistant
- KA QA Assistant

Purpose:
Combine all detailed assistant responsibilities into one unified execution mission for the next Keyboard Manager release.

This is non-breaking:
- no parser rewrite
- no MIDI rewrite
- no export rewrite
- no explorer rewrite
- build/smoke required
