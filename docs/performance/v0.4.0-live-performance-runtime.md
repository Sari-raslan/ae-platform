# v0.4.0 Live Performance Runtime

Adds:
- keyboard split engine (left/right hand zones)
- left-hand chord zone detection
- right-hand lead zone
- major/minor/7th chord detection
- realtime performance runtime
- tempo controls (40-240 BPM)
- transpose/octave foundation
- live performance panel with test events

This milestone turns MIDI events into arranger performance state.

## Features

### Keyboard Split Engine
- Configurable split point (default: C3 = 60)
- Separate left/right hand note tracking
- Real-time note on/off handling

### Chord Detection
Detects:
- Major (C, E, G)
- Minor (C, Eb, G)
- 7th (C, E, G, Bb)
- m7 (C, Eb, G, Bb)
- maj7 (C, E, G, B)
- Diminished (C, Eb, Gb)
- Augmented (C, E, G#)
- sus4 (C, F, G)

### Live Performance State
- Start/stop recording
- Real-time MIDI feed
- Chord detection from left hand
- Performance event history (64 events)
- Tempo, transpose, octave controls

## Usage

```javascript
import { createLivePerformanceRuntime } from "./livePerformanceRuntime.js";

const runtime = createLivePerformanceRuntime();

runtime.start();
runtime.feedMidi({ type: "note-on", note: 48, velocity: 100 });
runtime.setTempo(110);

const state = runtime.snapshot();
// {
//   running: true,
//   currentChord: "C major",
//   tempo: 110,
//   performanceEvents: [...]
// }
```
