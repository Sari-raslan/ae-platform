# Universal Arranger OS  Pro Simulation Launch Mode

## v11.0.0

This release changes the product strategy:

## Public Download Mode
Users can download and run the full interface as Simulation Mode.

## Simulation Mode Allows
- Explore the full interface
- Test controls
- Preview workflow
- Try arranger panels
- Learn the system

## Simulation Mode Blocks
- Save project
- Export song
- Create production sets
- Render audio
- Export MIDI
- Commercial production workflow

## Pro Mode
Pro Mode unlocks real production features after license activation.

## Business Goal
Allow users to experience the full product before purchase, while protecting real output and production use.
