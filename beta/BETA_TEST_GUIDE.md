# Universal Arranger OS  Beta Test Guide

## هدف النسخة التجريبية

اختبار:
- Electron desktop runtime
- MIDI workflow
- Arranger playback
- Sample engine
- Project save/load
- Installer
- Portable EXE

## المطلوب من المجربين

1. تشغيل البرنامج
2. اختبار الصوت
3. تجربة MIDI USB keyboard
4. تجربة Start / Stop arranger
5. تجربة تحميل samples
6. إرسال الملاحظات

## أسئلة الملاحظات

- هل البرنامج فتح بدون شاشة بيضاء أو سوداء
- هل الصوت يعمل
- هل MIDI ظهر
- هل الواجهة مفهومة
- هل حصل crash
- ما أهم ميزة ناقصة
